[Claude Code](https://docs.claude.com/en/docs/claude-code) 是 Anthropic 官方 CLI，默认连 Anthropic 自家网关；通过两个环境变量就能让它走 DimiLinks。

## 一键导入（推荐）

最快的方式是用 [ccSwitch](https://github.com/farion1231/cc-switch)：

1. 安装并登录 ccSwitch。
2. 在 DimiLinks 控制台「API 密钥」页，对一把限定 Claude 模型的密钥点「导入到 ccSwitch」。
3. ccSwitch 会自动新增名为 `Claude` 的 provider，base URL 已写为 `https://api-direct.dimilinks.com/`。
4. 在 Claude Code 启动前，运行 `cc-switch use <provider>` 即可切到 DimiLinks。

具体 deep link 结构见 [ccSwitch / Cherry Studio 一键导入](/api-reference/integrations/ccswitch)。

## 手动配置环境变量

如果你不用 ccSwitch，可以直接设置环境变量：

```bash
export ANTHROPIC_BASE_URL="https://api-direct.dimilinks.com/"
export ANTHROPIC_AUTH_TOKEN="sk-..."
```

随后启动 Claude Code：

```bash
claude
```

把变量写到 shell 配置（`~/.zshrc`、`~/.bashrc`）就能持久生效。如果你需要为不同密钥准备多个配置，可以借助 [direnv](https://direnv.net/)、[mise](https://mise.jdx.dev/) 之类的工具，按目录注入。

<Note>
  Claude Code 不会读取 OpenAI 兼容入口。务必使用 `https://api-direct.dimilinks.com/`，不要写成 `dimilinks.com/v1`。
</Note>

## 推荐模型

| 用途 | 模型 ID |
| --- | --- |
| 通用编码 / 重构 | `claude-sonnet-4-6` |
| 深度推理 / 复杂任务 | `claude-opus-4-7`、`claude-opus-4-7-thinking` |
| 大上下文长文档 | `claude-opus-4-7` |
| 低成本批量调用 | `claude-haiku-4-5-20251001` |

启动后用 `/model` 命令切换；模型 ID 与 [List models](/api-reference/models) 返回的一致。

## 常见问题

- **`401 Invalid token`**：环境变量没生效或密钥写错了；用 `echo $ANTHROPIC_AUTH_TOKEN` 自检。
- **`403 model_not_allowed`**：当前密钥的「模型限制」没有放开你想用的 Claude 模型，去控制台调整密钥即可。
- **请求一直 timeout**：先用 [Authentication](/api-reference/authentication) 里的 `curl` 简版排查；若 `curl` 通而 Claude Code 不通，多半是本地代理或 corp VPN 拦截。
- **想同时保留 Anthropic 官方账号**：用 ccSwitch 维护多 provider，运行前 `cc-switch use anthropic` 即可一键切回。
