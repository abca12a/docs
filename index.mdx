DimiLinks 提供与 OpenAI、Anthropic 使用习惯一致的统一 API。一把 Bearer 密钥就能调用 Claude、GPT、Grok 系列文本模型，并提交 `gpt-image-2` 图片任务和 `grok-imagine-video` 视频任务。

<Note>
  图片和视频生成都建议走异步链路：提交任务后用任务接口轮询结果。
</Note>

## Base URL

```text
https://dimilinks.com/v1
```

## 推荐链路

```text
POST /v1/chat/completions          # 对话（OpenAI 兼容，推荐编程使用）
POST /v1/messages                  # 对话（Anthropic 原生协议）
POST /v1/images/generations?async=true   # 图片：异步提交
POST /v1/videos/generations              # 视频：异步提交（默认即异步）
GET  /v1/tasks/{task_id}                 # 图片 / 视频任务通用查询
```
