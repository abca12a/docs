DimiLinks 提供兼容 OpenAI Images 使用习惯的图片 API。你可以用服务端 API Key 提交文生图、图生图和图片编辑请求，并通过任务接口稳定获取结果。

<Note>
  推荐使用异步链路：提交任务后轮询任务状态。图片生成通常需要几十秒到数分钟，异步方式更适合线上应用。
</Note>

## 推荐接入方式

```text
POST /v1/images/generations?async=true
GET  /v1/tasks/{task_id}
```

## Base URL

```text
https://lmage2.dimilinks.com/v1
```

<Warning>
  域名是 `lmage2.dimilinks.com`，首字符是小写字母 `l`，不是大写 `I`。
</Warning>
